🌍 Global Data-related Job Analysis & Salary Prediction with Machine Learning

📌 Project Description

This project analyzes worldwide data-related job listings and predicts salaries based on multiple factors. It combines web scraping, machine learning, and an interactive Streamlit web application.

🎯 Objectives

  • Scrape job data using Adzuna API
  • Visualize salary insights with Tableau
  • Train ML models for salary prediction
  • Build a user-friendly Streamlit web app

🛠 Technologies & Tools

  • Python (Pandas, NumPy, Scikit-learn, BeautifulSoup)
  • Web scraping via Adzuna API
  • ML: Linear Regression, Random Forest, XGBoost
  • SQL for data storage
  • Tableau dashboards
  • Streamlit web application

📊 Key Features

  • Automated data extraction from API
  • Interactive dashboards showing salary insights
  • ML-based salary predictions by experience, location, and sector
  • Streamlit app for real-time exploration
