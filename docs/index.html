🏠 House Prices - Advanced Regression Techniques

📌 Project Overview

This project was part of a Kaggle competition aimed at predicting house sale prices using 79 variables. The goal was to use machine learning and advanced regression techniques to build an accurate model.

📊 Dataset

  • 1,460 training samples with `SalePrice` as target
  • 79 explanatory features (e.g., size, location, materials)
  • Test set with 1,459 entries

🛠 Approach

  1. EDA: data cleaning, correlation analysis
  2. Feature Engineering: encoding, scaling, new variables
  3. Modeling: Ridge, Lasso, Random Forest, XGBoost, CatBoost
  4. Evaluation: RMSE, ensemble methods

🚀 Results

Best Model: CatBoost Regressor with RMSE of 0.1216

Top Features: NEW_TotalSF, OverallQual, GrLivArea, YearBuilt

📌 Conclusion

  • Feature engineering was critical for performance
  • Regularization reduced overfitting
  • Ensemble models like XGBoost gave the best results
