🌍 Global Data-related Job Postings & Salary Prediction with ML

📌 Project Description

This project analyzes global job offers related to data roles and predicts salaries based on multiple features. It integrates web scraping, machine learning, and a Streamlit web app for interactive use.

🎯 Objectives

  • Collect and analyze job data using the Adzuna API
  • Visualize data with Tableau (e.g., salaries by region, top skills)
  • Build ML models to predict salary
  • Deploy a Streamlit web app for salary prediction

🛠 Technologies & Tools

  • Python (Pandas, NumPy, Scikit-learn, BeautifulSoup)
  • Adzuna API for job data scraping
  • Machine Learning: Linear Regression, Random Forest, XGBoost
  • SQL database for data storage
  • Tableau for dashboards
  • Streamlit for web app

📊 Features

  • Automated job data extraction
  • Interactive salary dashboards
  • ML-based salary prediction based on experience, location, and industry
  • User-friendly Streamlit web app

🏠 House Prices - Advanced Regression Techniques

📌 Project Overview

This project analyzes the Kaggle competition "House Prices - Advanced Regression Techniques", focusing on predicting house sale prices based on various features.

📊 Dataset

  • 79 explanatory variables describing various aspects of a house (e.g., size, location, quality of materials).
  • Training data: 1,460 entries with known house prices.
  • Test data: 1,459 entries without price information.
  • Target Variable: SalePrice (house sale price).

🛠 Approach

  • Exploratory Data Analysis (EDA)
  • Feature Engineering (categorical to numerical, scaling, normalization)
  • Model Selection & Training (Linear Regression, Random Forest, XGBoost, etc.)
  • Evaluation & Optimization (RMSE, Feature Importance)
